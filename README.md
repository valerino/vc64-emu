# vc64-emu
my attempt at a portable Commodore64 emulator.

* just started!
* later in development may be splitted into 3 different repos: 
    * _v65xx_ (6502/6510 emulator)
    * _vc64_ (the c64 emulator itself)
    * _emushared_ (the shared utilities)
